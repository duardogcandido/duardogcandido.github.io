  • Avac Report
    Esta aplicação tem um código privado e foi desenvolvida para a empresa Avac Brasil. O sistema é utilizado para gerar relatórios de serviços de manutenção para equipamentos de ar condicionado, com a possibilidade de exportar os relatórios em PDF.
    Os Usuários podem criar, editar e excluir relatórios, além de poderem visualizar os relatórios de outros usuários.
    O sistema tem funções de notificações e agendamento de serviços, ao utilizar a agenda o usuário destinado ao serviço recebe notificações a cada 24h, 3h e 1h antes do agendamento como lembrete.
    Utilizando a função de Storage do Firebase também é possivel anexar documentos aos usuários para que utilizem quando estiverem em campo, sendo, documentos pessoais ou outros importantes para a execução do serviço ou assinatura de ordens.
    Apesar de ter sido criado como uma aplicação mobile hibrída, graças as tecnologias do Flutter, foi possivel utilizar o mesmo código para criar uma versão web da aplicação. A versão web é utilizada para gerenciamento dos relatórios, edições e controle dos usuários.

    Tecnologias utilizadas: Flutter, Dart, Material Design, Firebase, SqLite, Provider.
  • Click Saber/Acerta Brasil
    A plataforma Click Saber foi desenvolvida para a empresa Saber Educação, com o objetivo de fornecer um ambiente para correções de provas e simulados SAEB.
    Pelo site é possivel realizar a criação de provas e simulados, com a possibilidade de cadastrar o número de questões e o gabarito de resultados. Quando a prova é aplicado ao aluno, o professor pode realizar a correção do cartão respostas de maneira manual ou automâtica utilizando a função de leitura do gabarito que é feita através da câmera do dispositivo móvel.
    O projeto também tem funções de relatórios de desempenho dos alunos, baseado em habilidades SAEB ou BNCC com gráficos de desempenho e validações de quantos alunos acertaram, erraram ou não preencheram aquela questão.
    Este é um "case" de sucesso dentro da empresa, pois o projeto foi criado apenas como uma ideia de inovação e hoje é um dos produtos mais vendidos e utilizados, gerando um grande lucro para a empresa com os contratos de licença de uso.

    Tecnologias utilizadas: Flutter, Dart, Material Design, SqLite, Provider, Google Machine Learning, OpenCV, DartFFi, Leitor de QR Code.
